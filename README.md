## 数据来源

### 股票行情

- **数据源**：akshare（免费，无需注册）
- **数据类型**：后复权日度行情
- **字段**：日期、开盘价、收盘价、最高价、最低价、成交量、成交额
- **时间范围**：2020-01-01 至今

### 市场指数

- **沪深300（000300）**：CAPM分析的市场基准，代表A股大盘股
- **中证500（000905）**：代表A股中小盘股票，与沪深300形成互补

### 宏观指标

| 指标        | 来源    | 选择理由                                 |
| ----------- | ------- | ---------------------------------------- |
| CPI同比增速 | akshare | 反映通货膨胀水平，影响货币政策和股票估值 |
| M2同比增速  | akshare | 反映货币供应量变化，与股市流动性密切相关 |

### 财务数据

- **数据源**：akshare
- **指标**：ROE（净资产收益率）、净利润率
- **格式**：长格式（code, year, indicator, value）

## 存储方式

### 基础方式：CSV（方式A）

- 所有原始数据以CSV格式存储于 `data/`目录
- 合并后的综合数据保存为 `data/combined/combined_data.csv`

**CSV格式的优点**：

1. 通用性强，几乎所有数据分析工具都支持
2. 可读性好，可直接用文本编辑器查看
3. 跨平台兼容，适合数据交换

**CSV格式的局限**：

1. 文件体积较大，不适合大数据存储
2. 不保留数据类型信息，读取时需手动转换
3. 不支持列式读取，加载大文件效率低

### 进阶方式：Parquet（方式B）

- 清洗后的数据额外保存为Parquet格式
- 存储路径：`data/clean/stock_clean.parquet`

**选择Parquet的理由**：

1. **列式存储**：分析型查询效率高，可只读取需要的列
2. **高压缩率**：文件体积比CSV小约58%
3. **类型安全**：保留Schema信息，读取时自动恢复正确类型
4. **大数据友好**：与Spark、Hadoop等大数据工具无缝集成

## 如何运行

```bash

```

```bash

```

或在Jupyter中打开并运行所有单元格。

### 3. 运行数据清洗

```bash

```

### 4. 运行分析

```bash

```


```bash

```

**关键发现**：

- **收益最高**：比亚迪（30.31%），受益于新能源汽车行业高速发展
- **收益最低**：万科A（-17.99%），受房地产行业下行影响
- **波动最小**：工商银行（10.23%），防御性最强
- **回撤最小**：工商银行（-14.39%），抗风险能力最强

| 五粮液   | 白酒   | 0.00009  | 0.839  | 1.2488 | [1.18, 1.32]  | 0.5517 |

**关键发现**：

- **β>1（周期性）**：比亚迪、五粮液、中兴通讯，市场敏感度高
- **β<0.5（防御性）**：工商银行、中国石油，市场敏感度低
- **R²最高**：五粮液（0.55），与市场相关性最强
- **R²最低**：工商银行（0.11），受特有因素影响大

## 技术要点

1. **数据获取**：使用akshare免费接口，自动记录下载日志
2. **数据清洗**：完成缺失值处理、日期格式统一、重复值删除、离群值标注
3. **数据转换**：实现宽表与长表互转，处理不同频率数据合并
4. **数据存储**：同时支持CSV和Parquet格式
5. **可视化**：使用matplotlib和seaborn绘制专业图表
6. **回归分析**：使用statsmodels进行CAPM模型估计
