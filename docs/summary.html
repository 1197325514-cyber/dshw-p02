5  总结与结论

5.1 项目回顾

本项目完成了A股市场10只股票的数据获取、清洗、存储与分析工作，涵盖以下环节：

5.1.1 数据获取

  • 股票数据：10只A股股票，覆盖银行、汽车、房地产、白酒、能源、通讯、物流等7个行业
  • 指数数据：沪深300、中证500
  • 宏观数据：CPI同比增速、M2同比增速
  • 财务数据：ROE、净利润率

5.1.2 数据清洗

完成了缺失值处理、日期格式统一、重复值删除、离群值标注等清洗工作，并实现了宽表与长表转换。

5.1.3 数据存储

同时使用CSV和Parquet格式存储数据，Parquet格式体积压缩率达58%。

5.1.4 描述性统计

计算了年化收益率、年化波动率、偏度、峰度、最大回撤等统计量，发现：

  • 比亚迪年化收益最高（30.31%）
  • 万科A年化收益最低（-17.99%）
  • 工商银行波动最小、回撤最小

5.1.5 可视化分析

绘制了5张专业图表：

  1. 归一化收盘价走势图
  2. 日收益率分布图
  3. 相关系数热力图
  4. CPI与股市关系图
  5. Beta系数点图

5.1.6 CAPM回归

估计了10只股票的Beta系数，发现：

  • 3只股票β>1（周期性）：比亚迪、五粮液、中兴通讯
  • 7只股票β≤1（防御性）：工商银行、中国石油等

5.2 主要结论

5.2.1 行业特征

行业 风险特征 投资建议
银行 低β、低波动、低回撤 防御性配置，稳健收益
汽车 高β、高波动 牛市配置，风险较高
白酒 β适中，收益稳健 长期配置，消费龙头
房地产 收益为负，回撤大 谨慎配置，关注政策
能源 低β，受外部因素影响 分散化配置

5.2.2 投资启示

  1. 分散化投资：跨行业配置可以有效分散风险
  2. 周期把握：牛市配置周期性股票，熊市配置防御性股票
  3. 风险控制：关注最大回撤，设置止损点
  4. 长期视角：龙头股长期表现更优

5.2.3 方法论总结

  1. 数据获取：akshare免费易用，适合学术研究和个人投资分析
  2. 数据清洗：规范的数据清洗流程是分析质量的基础
  3. 数据存储：Parquet格式适合大数据场景
  4. 分析框架：CAPM是分析系统风险的有效工具

5.3 局限性与展望

5.3.1 局限性

  1. 样本有限：仅10只股票，代表性有限
  2. 时间范围：仅5年数据，未能覆盖完整经济周期
  3. 模型简单：仅使用单因子CAPM模型
  4. 宏观分析：仅分析了CPI一个宏观指标

5.3.2 未来展望

  1. 扩展样本：增加更多股票和行业
  2. 多因子模型：引入Fama-French三因子等模型
  3. 宏观分析：引入更多宏观经济变量
  4. 机器学习：尝试机器学习方法预测收益
